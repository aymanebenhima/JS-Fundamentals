# JavaScript Fundamentals

A collection of JavaScript fundamentals and examples to learn core concepts.

## Contents

- `0-javascript_is_amazing.js` - Print "JavaScript is amazing"
- `1-multi_languages.js` - Print three lines about programming languages
- `2-arguments.js` - Check number of command line arguments
- `3-value_argument.js` - Print first argument or "No argument"
- `4-concat.js` - Concatenate two arguments with " is "
- `5-to_integer.js` - Convert first argument to integer or print "Not a number"
- `6-multi_languages_loop.js` - Print three lines using array and loop
- `7-multi_c.js` - Print "C is fun" x times
- `8-square.js` - Print a square of X characters
- `9-add.js` - Add two integers using a function
- `10-factorial.js` - Compute factorial recursively

## Getting Started

1. Clone this repository
2. Run any JavaScript file using Node.js, example:
   ```bash
   node 0-javascript_is_amazing.js
   ```

## Prerequisites

- Node.js installed on your system

## Learning Path

This repository contains JavaScript exercises and examples organized by difficulty and topic. Each file demonstrates specific JavaScript concepts and best practices.
