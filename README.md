# JavaScript Fundamentals

A collection of JavaScript fundamentals and examples to learn core concepts.

## Contents

- `0-javascript_is_amazing.js` - Basic JavaScript variable declaration and console output

## Getting Started

1. Clone this repository
2. Run any JavaScript file using Node.js:
   ```bash
   node 0-javascript_is_amazing.js
   ```

## Prerequisites

- Node.js installed on your system

## Learning Path

This repository contains JavaScript exercises and examples organized by difficulty and topic. Each file demonstrates specific JavaScript concepts and best practices.
